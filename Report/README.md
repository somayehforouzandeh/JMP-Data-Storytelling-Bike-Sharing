This folder contains the final analytical reports for the bike-sharing demand analysis project developed using JMP.

## 📄 Reports

### 🇬🇧 English Report

**Data Storytelling with JMP**

This report presents a visual and analytical exploration of bike-sharing demand, focusing on:

- User type behavior
- Seasonal demand patterns
- Working days versus non-working days
- Weekly demand patterns
- Weather conditions
- Temperature
- Humidity
- Wind speed
- Regression analysis of environmental drivers

## 🔍 Key Analytical Findings

The analysis highlights clear differences between Registered and Casual users.

- Registered users demonstrate more stable and routine-oriented demand.
- Casual users are more sensitive to seasonality and environmental conditions.
- Temperature is the strongest environmental driver among the variables analyzed.
- Humidity has a meaningful secondary relationship with demand.
- Wind speed shows the weakest explanatory power among the environmental variables examined.
